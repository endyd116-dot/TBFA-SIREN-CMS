시스템 자동 발송 관리

회원에게 자동 발송되는 시스템 메시지(정기 결제 안내·카드 만료·자격 심사 결과 등)를 한 곳에서 관리합니다.
각 이벤트의 본문을 편집하고, 채널(이메일·SMS·카카오·인앱)별 on/off를 토글할 수 있습니다.

변경한 본문은 다음 자동 발송부터 적용됩니다. 변수({{name}} 등)는 발송 시 실제 값으로 자동 치환됩니다.
