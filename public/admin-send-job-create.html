새 발송 만들기

템플릿을 불러온 뒤 제목·본문을 그대로 보내거나 이번 발송에 한해 수정해서 보낼 수 있습니다. 템플릿 원본은 변경되지 않습니다. 채널은 여러 개 동시 선택 가능합니다.

발송 정보
관리자 식별용 — 수신자에게 노출되지 않습니다. 채널을 여러 개 고르면 각 채널별로 자동으로 (이메일)/(SMS) 등 접미사가 붙습니다.
활성 상태인 템플릿만 표시됩니다.
템플릿 선택 시 기본 채널이 자동으로 체크됩니다. 여러 채널을 동시에 선택하면 각 채널마다 개별 발송 작업이 등록됩니다.
SMS·카카오·인앱은 본문 그대로 발송됩니다. 이메일에만 적용.
첨부파일은 이메일에만 전송됩니다. SMS·카카오·인앱은 첨부 무시. 단일 파일 20MB 이하.
활성 상태인 그룹만 표시됩니다. 발송 시작 시점의 회원이 스냅샷됩니다.
현재 시각으로부터 1분 이상 미래여야 합니다.
