영수증 · 기부금 영수증

결제 직후 영수증

  • 입력한 이메일로 PDF 자동 발송
  • NotoSansKR 폰트 임베드 (한글 깨짐 없음)
  • 회원은 마이페이지에서 상시 다운로드

연말 기부금 영수증

  • 매년 1~2월 일괄 발급
  • 가입 시 입력한 주민번호 기준
  • 미입력 회원은 마이페이지에서 사전 등록 필요

카드 유효기간 등록 (정기·빌키 카드 후원)

카드로 정기 후원(빌키 등록)을 마치면, 결제 완료 화면에서 카드 유효기간(MM / YY)을 한 번 입력하는 칸이 나옵니다. 선택 입력이지만 등록해 두시면 좋습니다.

  • 입력 위치: 후원 결제가 끝난 완료 화면의 "카드 유효기간 등록" 칸 (예: 12 / 27)
  • 등록하면 카드 만료 30일·14일 전에 갱신 안내를 미리 보내드려 정기 후원이 끊기지 않습니다.
  • 카드번호·CVC가 아닌 만료 월/년만 입력합니다. 일시 후원·계좌이체에는 해당 없습니다.
  • 나중에 카드를 바꾸면 마이페이지에서 카드를 재등록할 때 새 유효기간으로 갱신됩니다.

신고 내용을 수정할 수 있나요?운영자 검토 전까지 본인 수정 가능. 검토 시작 후엔 보완 요청에 답변하는 방식.
AI가 외부 서비스로 분석한다는데?Google Gemini로 전송되지만 개인 식별 정보는 사전 제거. 회원이 동의해야 분석.
법률 상담 비용은?협의회 지원금 범위 내 무료. 초과 가능성 시 사전 안내.
운영자가 내 신원을 마음대로 볼 수 있나요?아니요. 별도 권한 + 사유 입력 + 감사 로그 영구 기록 필수. 부적절한 식별은 사후 추궁됩니다.
신고를 취소하고 싶어요/my-reports.html에서 [취소]. 운영자 검토 시작 후엔 제한될 수 있음.

온라인 추모관

협회가 추모하는 선생님들을 함께 기억하는 공간입니다. 상단 메뉴 추모관 > 온라인 추모관 또는 홈의 온라인 추모관 카드로 들어갑니다. (경로: /memorial.html)
