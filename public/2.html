LAYER 1 배치 실행

여러 작업을 한 명령으로 순서대로 실행. "메일 보내고, 카드 완료하고, 공지 올려줘"

✓ 최대 10개 도구 순차 실행
✓ 각 단계 성공 확인 후 다음 진행
✓ 실패 시 자동 롤백 안내
LAYER 2 파이프라인

조회 → 필터 → 처리의 3단계 흐름을 AI가 자동으로 연결. 결과를 다음 도구의 입력으로

✓ 회원 조회 → 필터 → 발송 자동 연결
✓ 신고 분석 → 담당자 배정 → 알림
✓ 지출 업로드 → AI 분류 → 회계 반영
