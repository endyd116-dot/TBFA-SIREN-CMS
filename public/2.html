01 — 업무 분산
후원·회원·신고·지원이 각각 다른 툴

Excel, 카카오폼, 이메일, 구글 시트 … 데이터를 모아 정리하는 데만 하루가 사라집니다.

02 — 수작업 과다
매월 보고서 · 영수증 · 메일 발송에 수십 시간

담당자 1명이 반복 업무에 묶여 정작 수혜자 지원에 시간을 쓰지 못합니다.

데이터는 쌓이지만 인사이트가 없음

Layer 1 & 2 — 실행 자동화

LAYER 1 배치 실행

여러 작업을 하나의 명령으로 순서대로 실행합니다. 예: "메일 보내고, 카드 완료하고, 공지 올려줘"

✓ 최대 10개 도구 순차 실행
✓ 각 단계 성공 확인 후 다음 진행
✓ 실패 시 자동 롤백 안내
LAYER 2 파이프라인

조회 → 필터 → 처리의 3단계 흐름을 자동으로 연결합니다. 이전 단계 결과가 다음 도구의 입력으로 이어집니다.

✓ 회원 조회 → 필터 → 발송 자동 연결
✓ 신고 분석 → 담당자 배정 → 알림
✓ 지출 업로드 → AI 분류 → 회계 반영

Layer 3 & 4 — 지능 & 자율화

LAYER 3 힌트 시스템

현재 작업이 완료되면 맥락에 따라 관련된 후속 작업을 자동으로 제안합니다.

"이번 달 가입 추이 브리핑을 확인해보시겠어요?"
LAYER 4 예약 & 자율 실행

"매주 월요일 오전 9시에 후원 현황 요약해줘" — 한 번 예약하면 설정한 일정에 따라 자동으로 반복 실행합니다.

자주 묻는 질문

도입 결정 전 주요 문의사항

Q
AI 분류가 틀리면 어떻게 되나요?
AI 분류 결과는 반드시 운영자 승인 단계를 거칩니다. 평균 정확도는 90%이며, 오분류 항목은 담당자가 검토·수정한 뒤 실행합니다. AI는 분류를 보조하고, 최종 판단은 담당자가 합니다.
Q
후원자 개인정보는 안전한가요?
전송·저장 전구간 암호화, JWT httpOnly 쿠키, 역할기반 접근제어, 감사 로그 1년 보관. 개인정보보호법 기준을 상회하는 아키텍처를 기본 적용합니다.
Q
기존 Excel·구글 시트 데이터는 이전되나요?
100% 이전 보장합니다. 회원·후원·지원 이력 전체를 SIREN으로 안전하게 옮기며, 이전 과정 전체를 저희가 책임지고 진행합니다.
Q
정말 월 구독료 없이 자급 구조인가요?
노션·슬랙 등 외부 협업툴 구독료는 ₩0입니다. 서버·DB·AI API 등 인프라 운영비는 정액으로 월 운영비에 포함됩니다. 예상 외 추가 과금 항목이 없습니다.
Q
담당자가 바뀌면 어떻게 하나요?
담당자 교체 시 무상으로 재교육을 진행합니다. 영상 매뉴얼과 문서 자료를 상시 제공하며, 주요 업무 기능은 교육 당일 내에 사용할 수 있도록 구성되어 있습니다.

💬

🔔
인앱 알림
