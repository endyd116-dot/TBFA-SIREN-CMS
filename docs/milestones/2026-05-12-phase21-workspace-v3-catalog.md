## 1. 합의 배경

워크스페이스 v2 (`backup-ws-v2-before-rollback` 브랜치 보존) 작업이 "기능이 따로 놀고 모달이 호환 안 됨"으로 롤백됨. v3는 **WBS(칸반)↔워크툴 완벽 연동**을 최우선 원칙으로 처음부터 재설계.

핵심 변화:
- v2 코드 재활용 **금지** (보존만)
- 통합 작업 모달로 모든 화면(워크툴·WBS·캘린더)이 동일한 카드 생성·수정 UX
- 칸반→WBS 전면 리네이밍
- 서비스 접수 즉시 자동 카드 생성 (R&R 1차+백업)
- 카드 이관(토스) + 이력 + 워처
- 메모-캘린더 연동(메모에 시간 필드 + 통합 조회)

## 2. 점검 결과 — 이미 있음 vs 미구현

### ✅ 이미 있어서 재활용·연동만
- 카드 상세 모달 (체크리스트 JSONB·첨부·댓글·@멘션·우선순위 배지·마감 임박 자동 색상·progress 슬라이더·북마크·보류 사유)
- 카드 액션 (생성·수정·드래그·삭제·아카이브)
- DB: `workspaceTasks`, `~Comments`, `~Attachments`, `~Reports`, `~Templates`, `~Notifications`(테이블만), `~ActivityLog`, `workspaceMemos`, `workspaceEvents`, `workspaceFiles`
- API: tasks/comments/attachments/reports/templates/events/memos CRUD
- 사이드바: 워크스페이스 1뎁스 + 5개 2뎁스(워크툴/칸반/캘린더/템플릿/파일함)

### ❌ 미구현 (신규 개발 대상)
1. 워처(관찰자) — 테이블·API·UI 전부 없음
2. 카드 이관(토스) — 테이블·API·UI 전부 없음
3. 할당한 작업 탭 — `assignedToMe`만 있고 `assignedByMe` 필터 없음
4. 알림 API + 드롭다운 UI
5. 서비스↔카드 동기화 — `lib/workspace-sync.ts` 없음, 4종 hook 없음
6. R&R 매핑 — `service_rnr` 테이블·API·탭 모두 없음
7. 운영자 부재 토글 — `admin_users`에 필드 없음
8. 메모 시간 옵션 — `workspaceMemos`에 `eventDate`/`eventTime` 없음
9. 캘린더 보강 — YIQ·빈 셀 3옵션·메모 미러 없음
10. WBS↔워크툴 양방향 동기화 — BroadcastChannel·`#task` hash 핸들링 없음
11. 칸반→WBS 리네이밍 — 전역 텍스트
12. 사이드바 정리 — 내작업/지시함/일정/메모 1뎁스 노출 제거 (워크툴 메인에서만 접근)
13. 카드 모달 타임라인 활성화 — UI 자리 있으나 `workspaceActivityLog` 미연결
14. 활동 피드 자연어 강화 + 클릭 이동 + 시간 그룹핑
15. 업무 템플릿 10종 시드
16. 마감 임박 자동 알림 cron (24h·72h)
17. 자연어 검색 (AI)
18. 사용자별 기본 보기 모드

| # | 항목 | 상태 |
|---|---|---|
