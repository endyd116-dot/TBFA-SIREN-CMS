# 순직유족급여 신청서 작성 가이드

## 1. 신청서 종류

### 1.1 순직유족보상금 청구서
- 일시금 지급 (기준소득월액의 23.4배 또는 35.1배)
- 인사혁신처 서식 제1호 사용

### 1.2 순직유족연금 청구서
- 매월 연금 지급 (기준소득월액의 38.5% 등)
- 인사혁신처 서식 제2호 사용

### 1.3 의료급여 청구서
- 순직으로 인한 치료비·요양비 소급 청구
- 인사혁신처 서식 제3호 사용

## 2. 청구서 핵심 항목 작성법

### 2.1 재해 개요 (사망 경위서)
작성 원칙:
- **육하원칙**: 언제, 어디서, 무슨 일로, 어떻게 사망했는가
- **시계열 기술**: 스트레스 누적 과정을 날짜순으로 정리
- **공무 관련성 강조**: 구체적 업무명, 민원 건수, 초과근무 시간 수치화
- **인과관계 서술**: 업무 스트레스 → 정신건강 악화 → 사망 연결

#### 작성 예시 구조:
```
1. 고인의 인적 사항 및 근무 이력
2. 재해 발생 경위
   - 근무 환경 (담당 학년·학급, 교과목, 담임 여부)
   - 업무 가중 내역 (초과근무, 추가 행정업무)
   - 민원·스트레스 사건 (구체적 날짜, 건수, 내용)
   - 정신건강 변화 (상담, 진료, 투약 시작 시점)
   - 사망에 이르는 과정
3. 업무 기인성 주장 근거
```

### 2.2 유족 인적 사항
- 대표 청구인 지정 (유족 중 1인)
- 전체 유족 목록 및 관계 기재
- 각 유족의 주민등록번호, 주소, 연락처

### 2.3 기준소득월액 확인
- 마지막 소속 기관(학교)에서 발급
- 사망 당시 월급여 기준 (각종 수당 포함)
- 교육청 인사기록 카드 참조

## 3. 사망 경위서 작성 시 주의사항

### 3.1 업무 관련성 입증을 위한 필수 기재 사항
- **초과근무 기록**: 구체적 날짜와 시간 (월별 집계)
- **민원 내역**: 접수 날짜, 민원인 유형, 내용 요약
- **심리적 충격 사건**: 구체적 사건과 날짜
- **건강 변화 시점**: 증상 발현 시점과 업무 사건의 시간적 근접성

### 3.2 흔한 실수 및 주의점
- 막연한 표현 금지 → "자주 야근했다" ❌ → "2023년 9월~11월 주 4회 이상 22시 이후 퇴근" ✅
- 감정적 서술 최소화 → 사실 중심 기술
- 의학 용어 주의 → 진단서 용어와 일치시킬 것
- 제3자 정보 포함 시 동의 확인

## 4. 첨부 서류 체크리스트

### 4.1 필수 서류
- [ ] 순직유족보상금 청구서 (인사혁신처 소정 양식)
- [ ] 사망진단서 원본 1통
- [ ] 가족관계증명서 (청구일 기준 3개월 이내 발급) 1통
- [ ] 주민등록등본 (유족 전원 포함) 1통
- [ ] 재직·경력증명서 (마지막 학교 발급) 1통
- [ ] 사망 경위서 (유족 자필 또는 날인)

### 4.2 업무 관련성 입증 서류
- [ ] 근무상황부·초과근무 기록
- [ ] 학교 내 민원 처리 대장
- [ ] 학교 카드 출퇴근 기록
- [ ] 이메일·메신저 기록 (업무 관련 스트레스 내용)
- [ ] 동료·관리자 진술서

### 4.3 의학적 서류
- [ ] 정신건강의학과 진료기록부 사본
- [ ] 처방전·투약기록
- [ ] 심리부검 소견서 (가능한 경우)
- [ ] 사망 전 마지막 진단서

### 4.4 선택 서류 (인정 가능성 제고)
- [ ] 유서 사본
- [ ] 개인 일기·메모 (업무 고충 기재 부분)
- [ ] SNS·메신저 대화 내용 (학부모·학교와의 갈등)
- [ ] 언론 보도 자료
- [ ] 탄원서 (동료 교사, 학교 학부모회 등)

## 5. 불인정 결정 대응 전략

### 5.1 심사청구 시 보완 자료
심사청구 단계에서 새로운 자료 추가 제출 가능:
- 추가 동료 진술서
- 심리부검 보완 자료
- 업무 가중 관련 교육청 공문
- 전문가 의견서 (정신건강의학과, 직업환경의학과)

### 5.2 행정소송 준비
- 변호사 선임 (교원단체 법률지원 활용)
- 공개 자료 확보 (교육청 정보공개청구)
- 유사 인정 사례 판례 수집
- 국가인권위원회 진정 병행 검토
