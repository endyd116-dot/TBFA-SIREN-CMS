# R42 SEO — 검색·공유·구조화·운영자립 (2026-05-28)

> 풀세트(검색기초+공유카드+구조화데이터+어드민 편집) + 동적 색인(캠페인·추모·소식·게시글) 자동 포함.
> 하이브리드 아키텍처: 고정 페이지=site_settings+빌드주입 / 동적 콘텐츠=Function 라우팅 / sitemap·robots·JSON-LD=별도.

## §2. API 명세

### 공개 (No auth)

| 함수 | 경로 | 동작 |
|---|---|---|
| `sitemap.ts` | `GET /sitemap.xml` | DB 동적 sitemap. Cache-Control public max-age=3600. |
| `page-with-seo.ts` | redirects 라우팅 `/{campaign,incident,activity,board-view,family-story,memorial-teacher}.html` | 정적 HTML 읽기 → URL query(slug/id)로 콘텐츠 메타 조회 → head 주입 → 반환. CDN 캐시 5분. |

### 어드민 (requireAdmin + canAccess('seo_edit'))

| 함수 | 메서드+경로 | 응답 키 |
|---|---|---|
| `admin-seo-list.ts` | `GET /api/admin-seo-list` | `{ ok, pages: [{path, title, hasDraft, lastUpdated}] }` |
| `admin-seo-get.ts` | `GET /api/admin-seo-get?path=` | `{ ok, path, published, draft, hasDraft }` |
| `admin-seo-save.ts` | `POST /api/admin-seo-save` | `{ ok, saved: true }` |
| `admin-seo-publish.ts` | `POST /api/admin-seo-publish` | `{ ok, published: N, buildTriggered: bool }` |
| `admin-seo-org.ts` | `GET/POST /api/admin-seo-org` | `{ ok, org: {...} }` / `{ ok, saved }` |
| `admin-seo-defaults.ts` | `GET/POST /api/admin-seo-defaults` | 동일 패턴 |

### 핵심 lib
- `lib/seo-meta.ts` — `getPageMeta(path, preferDraft)`, `getOrgMeta()`, `getDefaultMeta()`, `getContentMeta(table, key)` (동적 콘텐츠용)
- `lib/seo-injector.ts` — `injectMeta(html, meta)` — `</head>` 직전에 OG/Twitter/canonical 삽입, 기존 동일 태그 있으면 교체
- `lib/sitemap-builder.ts` — `buildSitemap()` — 발행 콘텐츠 7종 SELECT → URL+lastmod 배열

### Netlify Build Hook
- Swain이 Netlify 콘솔에서 Build Hook 생성 → env `NETLIFY_BUILD_HOOK_URL` 설정
- `admin-seo-publish.ts`가 publishDrafts() 후 이 URL에 POST → 빌드 시작
