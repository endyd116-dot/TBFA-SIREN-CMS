## 판단 원칙
- **화면 UI**(버튼·헤더·카드·탭·네비·배지·섹션타이틀): 아이콘 SVG로 치환. HTML은 `<span class="siren-icon-wrap" data-icon="이름"></span>`, JS 렌더는 `Icons.svg("이름")`.
- **텍스트 문맥**(HTML 속성값 placeholder/title/value, alert·toast·confirm 문자열, 로그): 이모지 제거 + 앞뒤 공백 정리. SVG 넣지 않음.
- **상태 이모지 의미 보존**: 🔴긴급→circle-dot(붉은 맥락), ✅완료→circle-check, ⚠경고→alert-triangle 등 같은 의미 아이콘.
- **미매핑**: 근접 아이콘(예 🥇🥈🥉→medal, 🔵🟢🟡🟠→circle-dot) 또는 순수 장식이면 제거.
