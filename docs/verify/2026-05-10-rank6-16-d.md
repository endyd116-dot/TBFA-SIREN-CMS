## 5. Swain 직접 클릭 검증이 필요한 항목

세션 쿠키 흐름이 필요한 부분. 다음 라운드에서 Swain이 직접 확인 권장:

1. **CSV 업로드 흐름**: 어드민 로그인 → 효성 자료 업로드 → 매칭 결과 표시 → pending 명단에서 1건 확정 → 후원 행 생성 확인
2. **D3·D4 화면**: 통합 회원 명단·정기 후원자 명단에서 효성 계약 컬럼(약정일·약정금액·결제수단·다음 결제일)이 자료대로 표시되는가
3. **D7 대시보드 KPI**: 6장 KPI 숫자 + 검증 Alert(미매칭·중복·전이) + 최근 import 이력이 표시되는가
4. **자동 상태 전이**: 다음 daily cron 후 잠재→정기 또는 정기→잠재 전이가 발생했는지 후원자 명단 분포로 확인

코드 정합성·라이브 진입·인증 가드는 본 보고서로 통과. 업무 시나리오 클릭 테스트는 Swain 직접.
