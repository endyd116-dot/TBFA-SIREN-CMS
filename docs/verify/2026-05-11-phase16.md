## 3. 발견·수정된 버그

### BUG-16-01 (High) — 기간 선택 버튼 미구현으로 Q2 동작 불가
- **증상**: Q2 "[90일] 클릭 → KPI 갱신" 시나리오가 동작하지 않음 — 대시보드 화면에 기간 선택 버튼이 없음
- **원인**: `renderShell`에 기간 선택 버튼이 없고, KPI 로드가 항상 `?period=30d`로 고정 호출됨
- **수정**: 헤더 우측에 [30일] [90일] [180일] [365일] 버튼 추가, 클릭 시 해당 기간으로 KPI 재호출
- **파일**: `public/js/admin-unified-dashboard.js`

### BUG-16-02 (High) — 연간 탭 클릭 시 분기 데이터가 반환됨
- **증상**: 대표 보고서 [연간] 탭 클릭 시 연간 집계가 아닌 분기 집계 반환
- **원인**: 프론트 탭 버튼 값은 `"yearly"`이나, 백엔드는 `type === "annual"`만 연간으로 인식. `"yearly"` 수신 시 `quarterly` 분기로 처리
- **수정**: `loadBoardReport` 함수에서 `yearly → annual` 매핑 추가 후 API 호출. 월간 탭(`monthly`)은 이 API 미지원이므로 안내 메시지로 대체
- **파일**: `public/js/admin-report.js`
