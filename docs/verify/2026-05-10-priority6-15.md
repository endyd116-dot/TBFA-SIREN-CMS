## 3. 검증 가이드 (Swain용)

1. 어드민 → CSV 업로드 탭 → 효성 수납 내역 CSV 또는 기업은행 CSV 파일 선택 → 업로드
2. 자동 파서가 파일 유형 감지 → 대기열 파싱 결과 표시 확인
3. 자동 매칭된 회원 목록 검토 → 매칭 점수·이유 확인
4. "확정" 클릭 → 후원 데이터 저장 확인
5. 같은 항목 재확정 시도 → 오류 메시지 확인 (멱등성 검증)
