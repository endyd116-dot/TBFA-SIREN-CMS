## A-1. 메인 (`/`)

**구성 요소** (어드민 > `사이트 빌더`에서 순서·노출 조정 가능):
- 상단 캠페인 배너 (1개 활성)
- 후원 진입 CTA
- 최근 활동·소식 3~6개
- 캠페인 카드 (진행률·모금액·D-day)
- 협의회 소개 요약
- 후원·신고·법률 상담 진입 버튼 3종

**방문자 흐름**:
1. 메인 진입 → 후원 CTA 클릭 → `/donate.html`
2. 메인 → 신고 진입 → `/incidents.html` 목록 → 글 작성 또는 `/report-harassment.html`
3. 헤더 [로그인] → 회원만 가능한 기능(글쓰기·후원 영수증·구독 관리) 활성화

## A-2. 회원가입·로그인

### 가입 (`/`에서 로그인 모달 진입 → "회원가입" 탭)
1. 이메일 입력 → **이메일 인증 코드** 발송 (Resend 사용, 5분 유효)
2. 코드 확인 후 비밀번호·이름·휴대폰 입력
3. **휴대폰 본인인증** 코드 발송 → 확인
4. 약관·개인정보 동의 → 가입 완료
5. 가입 시 자동으로 **포인트 적립**(가입 보상) 및 **알림 기본 설정** 적용

**가입 채널 추적**: URL 파라미터(`?src=캠페인A`)로 진입한 회원은 어드민에서 채널별 KPI로 분리 집계됩니다.

### 로그인
- 이메일/비밀번호 → JWT(httpOnly 쿠키, 7일 유효) 발급
- 로그인 실패 5회 → 일시 잠금
- **블랙리스트 회원**: 로그인은 되지만, 후원·신고·게시글 작성 등 핵심 기능 차단 (메시지: "귀하의 서비스가 차단되었습니다.")
- 비밀번호 재설정: 로그인 모달 > "비밀번호를 잊으셨나요?" → 이메일로 재설정 링크

### 회원 탈퇴
- `/mypage.html` 하단 [회원 탈퇴]
- 진행 중 정기후원이 있으면 **먼저 해지** 필요
- 탈퇴 후 30일 보관 → 영구 삭제

## A-3. 후원 (`/donate.html`)

### 결제 수단 3종

| 수단 | 일시 | 정기(자동) | 비고 |
|---|---|---|---|
| 토스페이먼츠(카드) | ✅ | ✅ (토스 빌링) | 즉시 결제·즉시 영수증 |
| 효성 CMS+ (계좌 자동이체) | ✅ | ✅ | 계좌 등록 후 매월 출금 |
| 계좌이체(무통장) | ✅ | ❌ | 입금 확인 후 영수증 발급 |

### 일시 후원 흐름
1. `/donate.html` 진입 → 금액 선택(1만/3만/5만/10만/직접입력)
2. 후원자 정보(비회원도 가능) — 이메일 영수증 발송
3. 결제 수단 선택 → 토스 결제창 또는 계좌이체 안내
4. 결제 성공 → 영수증 PDF 자동 발급, 회원이면 마이페이지 누적

### 정기 후원 등록
1. `/donate.html`에서 [정기 후원] 탭 → 금액·결제일·결제수단 선택
2. **토스 빌링**: 카드 등록(`/billing-register.html`) → 매월 지정일 자동 청구
3. **효성 CMS+**: 계좌·생년월일·인감 입력 → 효성 본인인증 → 매월 출금
4. 성공 후 `/billing-success.html`, 실패 시 `/billing-fail.html`

### 후원 영수증·기부금 영수증
- 결제 직후 이메일 자동 발송
- 마이페이지 > 후원 내역에서 다운로드 (PDF, NotoSansKR 임베드)
- 연말정산용 기부금 영수증: 가입 시 입력한 주민번호 기준 발급

## A-4. 캠페인 (`/campaigns.html`, `/campaign.html`)

- **목록**: 진행 중 / 종료 / 인기 필터
- **상세**: 모금액·목표·참여자 수·D-day·스토리(Toast UI Editor) + [참여하기] / [서명하기] / [후원하기]
- **참여 방식**: 캠페인마다 다름 — 단순 서명 / 모금 후원 / 댓글 응원 / 응답폼 제출
- **AI 카피**: 어드민이 캠페인 작성 시 AI가 카피 초안 생성(`admin-ai-copy`)

---

## A-5. 게시판 (`/board.html`)

- **권한**: 열람 비회원 가능 / 작성·댓글 회원만 / 본인 글만 수정·삭제
- **에디터**: Toast UI v3.2.2 (3-tier CDN 폴백) — 이미지 붙여넣기 자동 업로드(R2)
- **신고**: 부적절 댓글/글 [🚩 신고] → 어드민 > 신고 관리로 전달
- **검색**: 제목·내용·작성자

---

## A-6. 사건 제보 진입 (`/incidents.html`)

- 공개로 등록된 사건 목록 (어드민이 익명화 후 노출)
- 본인이 신고하려면 헤더 [신고하기] 또는 `/report-harassment.html` 진입 (자세한 절차는 [`siren-report.md`](siren-report.md))

## A-7. 법률 지원 안내 (`/legal-support.html`)

- 법률 상담 절차·지원 범위·전문 변호사 풀 안내
- [법률 상담 신청] 클릭 → 신고 폼으로 이동 ([`siren-report.md`](siren-report.md) §3 참고)

## A-8. 자료실·소식·활동

- 모두 어드민 콘텐츠 등록(`admin-resources`, `admin-activity-posts`, `admin-media-posts`) 기반
- 카테고리·태그·검색 지원
- 자료 다운로드는 R2 Pre-signed URL(만료 시간 있음)

---

## A-9. 약관·정책 페이지

- `/terms.html` 이용약관, `/privacy.html` 개인정보 처리방침, `/ethics.html` 윤리경영, `/email-reject.html` 이메일 무단수집 거부
- 모두 어드민 > 콘텐츠 관리(`admin-content-pages`)에서 수정 — 코드 배포 불필요

---

## A-10. 공통 UI

- **헤더 메뉴**: 어드민 > 메뉴 관리(`admin-nav-menus`)에서 순서·노출 제어
- **푸터**: 협의회 정보·연락처·SNS — 사이트 설정(`admin-site-settings`)
- **팝업**: 메인 진입 시 노출되는 공지/이벤트 팝업 — 어드민 > 팝업 관리
- **알림(웹·이메일)**: 회원 활동(댓글·신고 답변·후원 처리)별 — 마이페이지에서 수신 채널 제어

## A-11. 자주 묻는 질문 (방문자 관점)

| 질문 | 답 |
|---|---|
| 비회원도 후원할 수 있나? | 가능. 영수증은 입력한 이메일로 발송. |
| 후원 영수증을 다시 받으려면? | 회원이면 마이페이지 후원 내역에서 다운로드. 비회원은 협의회 문의. |
| 정기 후원 중단은? | 마이페이지 > 내 구독 관리 > [해지]. 즉시 다음 회차부터 미청구. |
| 게시판 글이 삭제됐어요. | 본인 또는 운영자 삭제. 운영자 삭제는 신고 접수에 따른 처리. |
| 신고하면 신원이 노출되나? | 익명 보호. 운영자도 별도 권한(익명 신원 식별 + 감사 로그) 없이는 못 봄. [`siren-report.md`](siren-report.md) §5 참고. |
