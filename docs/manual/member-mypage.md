# B. 회원 마이페이지 메뉴얼

> **대상**: 로그인한 일반 회원 (`/mypage.html` 진입)
> **권한**: 본인 데이터만 조회·수정 가능

---

## B-0. 페이지 한눈에

| 페이지 | 경로 | 무엇을 |
|---|---|---|
| 마이페이지 메인 | `/mypage.html` | 후원 내역·내 신고·포인트 요약 |
| 포인트·뱃지 | `/mypage-points.html` | 적립/사용/만료 + 획득 배지 |
| 포인트 랭킹 | `/ranking.html` | 회원 전체 랭킹 |
| 내 구독 관리 | `/my-subscriptions.html` | 정기 후원·뉴스레터 구독 |
| 받은 메시지 | `/my-send-history.html` | 운영자가 보낸 이메일·SMS 이력 |
| 알림 수신 설정 | `/settings-notifications.html` | 카테고리별 ON/OFF |
| 정기후원 카드 등록 | `/billing-register.html` | KICC 빌키 카드 등록 |
| 결제 결과 | `/payment-success.html`, `/payment-fail.html`, `/billing-success.html`, `/billing-fail.html` | 결제·정기등록 결과 안내 |

---

## B-1. 마이페이지 메인 (`/mypage.html`)

화면 구성:
- **상단 카드**: 등급(브론즈/실버/골드/플래티넘), 누적 후원금, 포인트 잔액
- **탭**:
  1. 내 후원 — 일시·정기 후원 내역, 영수증 다운로드
  2. 내 신고 — 사건·괴롭힘·법률 상담 본인 신고 상태
  3. 내 게시글 — 작성한 글·댓글
  4. 내 구독 — 정기 후원·뉴스레터
  5. 내 캠페인 참여 — 서명·참여한 캠페인
  6. **본인 명세서** — (운영자/간사 한정) 급여 명세서 본인 탭 (R37 추가)
- **하단 액션**: [개인정보 수정] [비밀번호 변경] [회원 탈퇴]

---

## B-2. 후원 내역 관리

### 일시 후원
- 결제일·금액·결제 수단·영수증 다운로드(PDF)
- 환불 요청은 어드민에 문의 (자동 환불 기능 없음)

### 정기 후원
- 현재 활성/일시정지/해지 상태
- 다음 결제 예정일 표시
- [해지] → 즉시 정기 등록 중단, 다음 회차부터 미청구
- [결제수단 변경] → KICC 빌키는 카드 재등록(`/billing-register.html`), 효성은 신규 신청 필요
- **카드 만료 임박 알림**: `cron-billing-card-expiry`가 매일 점검 → 만료 2개월 전부터 이메일·앱 알림 → 마이페이지에서 새 카드 등록

### 기부금 영수증
- 연 1회(1~2월) 일괄 발급 + 마이페이지 상시 다운로드
- 발급 기준: 가입 시 입력한 주민번호. 미입력 회원은 마이페이지 > 개인정보 수정에서 추가

---

## B-3. 포인트·뱃지 (`/mypage-points.html`)

### 포인트
- **적립**: 가입(웰컴), 후원(금액×규칙), 캠페인 참여, 게시글 추천 받기, 출석
- **사용**: 리워드 교환 (어드민 > 리워드 관리에서 정의된 보상 항목)
- **만료**: 적립일 기준 12개월 (정책은 어드민 > 포인트 규칙에서 조정)
- **수동 조정**: 운영자가 가산·차감하면 사유와 함께 이력에 표시

### 뱃지
- 자동 발급: "첫 후원", "10회 후원", "3년 연속 후원", "10명 추천" 등
- 어드민 > 게이미피케이션 관리에서 발급 규칙 정의
- 프로필 노출은 회원이 선택

### 등급 (Bronze/Silver/Gold/Platinum)
- 매일 새벽 `cron-grade-recalc`가 누적 후원금 기준으로 재계산
- 등급별 혜택은 어드민 > 등급 정의에서 설정

---

## B-4. 내 구독 관리 (`/my-subscriptions.html`)

- **정기 후원**: 금액·결제일·다음 청구일·상태 — 일시정지/해지/금액 변경
- **뉴스레터**: 카테고리(소식·캠페인·법률 이슈)별 구독
- **이탈 예측**: `cron-churn-predictor`가 후원 패턴 분석 → 이탈 위험 회원에게 운영자가 회복 캠페인 발송

---

## B-5. 받은 메시지 (`/my-send-history.html`)

- 운영자가 발송한 이메일·SMS·푸시 알림 이력
- 발송 시각·채널·내용·열람 여부 표시
- 어드민에서는 `admin-member-send-history`로 회원별 발송 현황 조회

---

## B-6. 알림 수신 설정 (`/settings-notifications.html`)

카테고리별 ON/OFF (이메일·웹·앱푸시 채널별 별도):
- 후원 처리 결과
- 정기 후원 결제 예정·완료
- 내 신고 상태 변경
- 캠페인 시작·종료
- 뉴스레터
- 댓글·답글 알림
- 시스템 공지

기본값은 어드민 > 알림 기본 정책(`admin-notification-defaults`)에서 정의.

---

## B-7. 결제 결과 페이지

| 페이지 | 언제 보이나 |
|---|---|
| `/payment-success.html` | 일시 후원 결제 성공 — 영수증 발송 안내 |
| `/payment-fail.html` | 결제 실패 — 사유·재시도 안내 |
| `/billing-success.html` | 정기 후원 카드 등록 성공 — 첫 청구 예정일 |
| `/billing-fail.html` | 카드 등록 실패 — 사유·재시도 |

모두 사용자에게 다음 액션을 안내하는 짧은 안내 페이지.

---

## B-8. 개인정보 보호

- **휴대폰 번호·주소**: 본인만 수정. 운영자는 발송·연락 목적 외 조회 안 함
- **주민번호**: 기부금 영수증 발급 목적으로만 저장(암호화). 본인 미입력 시 발급 불가
- **이메일 변경**: 새 이메일 재인증 후 적용
- **비밀번호**: 변경 시 기존 비밀번호 확인 필요

---

## B-9. 자주 묻는 질문 (회원 관점)

| 질문 | 답 |
|---|---|
| 후원 금액을 바꾸려면? | `/my-subscriptions.html` > 해당 정기 후원 > [금액 변경]. 다음 결제부터 반영. |
| 카드가 만료됐어요. | `/billing-register.html`에서 새 카드 등록. 자동으로 기존 정기 후원에 연결. |
| 포인트가 사라졌어요. | 적립일 12개월 경과 시 자동 만료. 만료 이력은 `/mypage-points.html` 하단에서 확인. |
| 운영자가 내 게시글을 삭제했어요. | 어드민 > 댓글/게시글 신고 처리에 따른 결과. 알림으로 사유 전달. |
| 신고 답변이 안 와요. | `/mypage.html` 내 신고 탭에서 상태 확인. 운영자 검토 중일 수 있음. |
| 받는 메일이 너무 많아요. | `/settings-notifications.html`에서 카테고리별 끄기. 법정 통지(결제 실패 등)는 끌 수 없음. |
